# Casos de Teste

Esta pasta contém casos de teste criados a partir de sistemas e sites reais, com o objetivo de simular situações do dia a dia de um QA Júnior.

Os casos de teste estão organizados por funcionalidade e cobrem cenários positivos e negativos, incluindo pré-requisitos, passos para execução e resultados esperados.

O foco é demonstrar a minha capacidade de estruturar testes claros, reproduzíveis e fáceis de compreender para programadores e outros membros da equipa.
