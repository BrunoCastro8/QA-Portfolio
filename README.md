# QA Portfolio — Bruno Castro

Este repositório foi criado como parte do meu processo de aprendizagem e transição para a área de Quality Assurance (QA).

Aqui organizo os meus estudos, resumos teóricos, casos de teste, relatórios de bugs e projetos práticos, com o objetivo de desenvolver uma base sólida em testes de software e boas práticas de qualidade.

## Estrutura do Repositório

- `fundamentos/` — Conceitos básicos de QA, tipos de teste e áreas da qualidade de software  
- `istqb/` — Resumos e apontamentos do estudo para a certificação ISTQB Foundation  
- `agile/` — Metodologias ágeis e papel do QA em Scrum e Kanban  
- `test-cases/` — Casos de teste criados a partir de sites e sistemas reais  
- `bugs/` — Relatórios de bugs simulados e documentados  
- `gestao_testes/` — Planos e relatórios de testes  
- `api/` — Estudo e testes básicos de APIs  
- `projeto_final/` — Projeto completo de QA, do planeamento ao relatório final  
- `entrevistas/` — Perguntas e respostas para preparação de entrevistas

## Objetivo

Construir um portefólio prático que demonstre os meus conhecimentos em QA Manual, fundamentos de teste de software e ferramentas utilizadas no mercado, com foco em aprendizagem contínua e qualidade de processos.
