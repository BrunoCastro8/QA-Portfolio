# Relatórios de Bugs

Esta pasta contém relatórios de bugs documentados com base em testes realizados em sistemas e sites reais.

Cada relatório inclui informações sobre o ambiente, severidade, prioridade, pré-condições, passos para reproduzir, resultado atual e resultado esperado.

O objetivo é demonstrar a minha capacidade de comunicar defeitos de forma clara e objetiva, facilitando a análise e correção por parte das equipas de desenvolvimento.
