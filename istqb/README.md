# ISTQB Foundation — Resumos de Estudo

Esta pasta contém os meus resumos e apontamentos pessoais do estudo para a certificação ISTQB Foundation.

O objetivo destes ficheiros é consolidar os fundamentos de testes de software, conceitos de qualidade, níveis e tipos de teste, bem como a diferença entre QA, QC e testes no ciclo de vida do desenvolvimento de software (SDLC).
