# Fundamentos de Quality Assurance (QA)

Esta pasta contém os meus apontamentos e resumos pessoais sobre os conceitos base de Quality Assurance e testes de software.

Aqui estão incluídas explicações sobre o que é QA, as diferenças entre QA, QC e QM, os tipos de teste, as áreas dentro da carreira de QA e o papel do QA no ciclo de vida do desenvolvimento de software (SDLC).

O objetivo destes ficheiros é consolidar a minha base teórica e desenvolver a capacidade de explicar conceitos de qualidade de forma clara e estruturada.
